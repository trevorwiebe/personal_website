Stanza

Description:

Stanza is a gospel sheet music app designed to bring gospel music into the modern age. It is being developed collaboratively with other designers and developers. Stanza offers an extensive library of gospel sheet music, featuring everything from classic hymns to contemporary worship songs. Its user-friendly interface ensures that both novice singers and experienced choir directors can navigate effortlessly, making it easy to find and sing your favorite songs.

What sets Stanza apart is its blend of tradition and technology. Built using the latest advancements in mobile app development, it leverages Kotlin for efficient performance and Jetpack Compose for a dynamic and intuitive user interface. Track-A-Cow

Description:

Track-A-Cow is a sophisticated mobile application, originally developed in Java and later updated to Kotlin, designed to streamline operations for a local feed-lot. This innovative app allows feed-lot managers to efficiently monitor and manage key aspects of their cattle operations, including tracking feed allocation, medication administration, and tracking mortality rates.

The app functions by initially loading a local database to display existing data. It then seamlessly connects to Firebase Realtime Database, enabling real-time synchronization across multiple devices. This means that when one user updates information, it instantly reflects on all connected signed in devices, facilitating collaborative cattle management with up-to-the-minute data.

Track-A-Cow showcases advanced programming skills in both Java and Kotlin, as well as expertise in creating responsive applications that enhance efficiency and data accuracy in the feed-lot industry. This project serves as a powerful demonstration of my ability to create practical working solutions.
